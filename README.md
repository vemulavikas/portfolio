# Portfolio

A simple, fast, responsive personal portfolio site.

## Run locally

You can open `index.html` directly in a browser, or run a local server.

### Option A: Python (recommended)

```bash
python -m http.server 5500
```

Then open `http://localhost:5500`.

### Option B: VS Code Live Server

Install the “Live Server” extension and click “Go Live”.

## Add your resume

- Export your resume as a PDF named `resume.pdf`
- Place it in the project root (same folder as `index.html`)
- The site will automatically show a Resume section with download + preview
